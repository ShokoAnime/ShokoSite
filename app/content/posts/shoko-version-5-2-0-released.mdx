Starting 2026, we're happy to announce that Shoko Server 5.2.0 is now available! This release focuses on
improving a lot of the underlying code, fixing bugs reported from version 5.1.0 and working towards making sure we are
ready for some of the bigger changes we have planned for 2026.

## Better support for TMDB

In version 5.2.0, we added more support for TMDB introduced full support for TMDB. With this
latest release, we've added preferred TMDb episode and season images and Enabled TMDB auto linking by default on new installations.

## Plugins

With version 5.2.0, we made cast, crew, studios & characters data accessible to plugins.
Additionally plugins will get events when images gets added/updated/removed on entities.
